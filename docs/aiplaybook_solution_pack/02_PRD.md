# 02 PRD

## 产品目标
构建一个可以在 Web 端直接交付价值并完成订阅变现的 AI 落地平台。

## 第一版功能范围
### 必做
1. 首页
2. Digest 列表/详情
3. Playbook 列表/详情
4. Templates 列表/详情
5. Tools 列表
6. 三个工具详情页
7. Pricing 页
8. 登录/注册
9. 会员订阅
10. Dashboard
11. 收藏
12. 咨询页
13. 邮件订阅入口

## 用户故事
### 访客
- 快速理解网站做什么
- 免费体验一个工具
- 看公开内容决定是否付费

### Pro 用户
- 访问所有 Playbook、模板和工具
- 收藏内容
- 查看工具历史
- 明确知道本周更新了什么

## 非功能需求
- 响应式
- 快速加载
- SEO 友好
- 低成本部署
- 组件化
- 适合 Cloudflare
