# 05 UI 风格指南

## 品牌感觉
- 清晰
- 专业
- 克制
- 执行导向
- 高密度但不拥挤

## 视觉关键词
- Editorial + Productized
- Minimal but premium
- Modern knowledge workspace

## 推荐色板
- Background: #0B1020
- Surface: #12192B
- Card: #182136
- Text Primary: #F4F7FB
- Text Secondary: #AAB4C5
- Brand: #6D7CFF
- Brand Hover: #8390FF
- Success: #2CCB8F
- Border: #24304A

## 字体
- Inter
- PingFang SC / Noto Sans SC

## 字号
- H1: 48-56
- H2: 32-36
- H3: 24-28
- Body: 16
- Caption: 14

## 布局
- 最大宽度 1200px
- 阅读区 760-820px
- 统一 8px spacing scale

## 圆角
- 卡片 20px
- 按钮 14px
- 输入框 14px

## 风格原则
不要炫技式 AI 视觉。
更像高质量内容平台 + 轻量产品后台。
