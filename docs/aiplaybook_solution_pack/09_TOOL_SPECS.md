# 09 工具规格

## 原则
- 只做结果明确的任务
- 不做聊天机器人
- 每个工具一屏解决一个问题
- 输出可复制

## Tool 1: Landing Page Generator
### 输入
- 产品名
- 面向人群
- 核心卖点
- 用户痛点
- 语气风格
### 输出
- Headline
- Subheadline
- Benefits
- CTA
- FAQ

## Tool 2: Content Repurposer
### 输入
- 原始内容
- 平台选择
- 风格
- 语言
### 输出
- X 帖子
- LinkedIn 帖子
- 小红书文案
- 邮件摘要

## Tool 3: SOP Builder
### 输入
- 流程名称
- 目标
- 输入项
- 限制条件
- 团队规模
### 输出
- SOP steps
- checklist
- risks
- success criteria
