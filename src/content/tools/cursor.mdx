# Cursor: The AI-first Code Editor

Cursor is an AI-first code editor built as a fork of VS Code. It provides native AI integrations that fundamentally change how developers write, debug, and refactor code.

## Key Features

- **Command K (Cmd+K)**: Inline code generation. Describe what you want, and Cursor writes the code.
- **Chat (Cmd+L)**: A ChatGPT-like interface that understands your entire codebase.
- **Codebase Indexing**: Cursor reads and understands your entire workspace, providing highly contextual answers.
- **Auto-debug**: Automatically finds and fixes errors in your terminal.

## Why use Cursor over VS Code + Copilot?

While GitHub Copilot is a great autocomplete tool, Cursor is an integrated AI experience. Copilot guesses your next few words; Cursor can architect entire files, understand context across multiple directories, and autonomously fix complex bugs.
