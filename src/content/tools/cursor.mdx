## 工具介绍

Cursor 是目前最强大的 AI 代码编辑器，基于 VS Code 开发，集成了深度的 AI 能力。

## 核心能力

- **Composer**: 整个项目的 AI 编辑模式。
- **Chat**: 随时询问代码逻辑。
- **Codebase Indexing**: 理解整个项目的上下文。

## 典型使用场景

1. **从零构建 MVP**: 用自然语言描述需求，让 AI 生成核心架构。
2. **重构代码**: 选中代码块，输入优化指令。

## 优点

- 上下文理解极强
- 与 VS Code 插件生态兼容
- 响应速度快

## 缺点

- Pro 版价格相对较高 ($20/mo)

## 价格说明

- **Free**: 限制 AI 使用次数。
- **Pro**: 无限次使用 Claude 3.5 Sonnet 等模型。
