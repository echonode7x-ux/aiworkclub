## 1. 什么是 Cursor?

Cursor 是一个分叉自 VS Code 的代码编辑器，它将 AI 能力（主要是 Claude 3.5 Sonnet 和 GPT-4o）深度集成到了编辑流程中。

## 2. 安装与初始配置

1. 访问 [cursor.com](https://cursor.com) 下载并安装。
2. 登录账号。
3. 导入 VS Code 配置（如果你以前用 VS Code）。

## 3. 核心功能操作

### Command + K: 行内编辑
选中代码，按下 `Cmd + K`，输入你的修改指令。

### Command + L: 聊天侧边栏
询问关于代码库的任何问题。

### Composer (Cmd + I)
进行跨文件的代码生成和重构。

## 4. 最佳实践

- **提供上下文**: 善用 `@Files` 和 `@Codebase`。
- **小步迭代**: 不要一次性让 AI 写太复杂的逻辑，分步执行效果更好。
