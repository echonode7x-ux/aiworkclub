# Landing Page Copywriting Framework

Creating a high-converting landing page requires a deep understanding of customer pain points and clear, concise messaging. Use this mega-prompt with Claude 3.5 Sonnet or GPT-4o to generate your first draft.

## The Mega-Prompt

```text
Act as a world-class direct-response copywriter. I need you to write the copy for a new landing page for my product.

Here is the context:
- Product Name: [Insert Name]
- Target Audience: [Insert Audience]
- Main Problem Solved: [Insert Problem]
- Core Unique Mechanism: [How your product works differently]
- Primary Call to Action (CTA): [e.g., Start 14-day free trial]

Please use the PAS (Problem-Agitation-Solution) framework.

Structure the output exactly like this:
1. **Hero Section**: A bold, outcome-driven H1 headline (max 8 words). A descriptive H2 subheadline (max 20 words). The primary CTA text.
2. **Social Proof Section**: Write a placeholder for a specific, realistic-sounding customer testimonial.
3. **Problem Agitation**: 3 bullet points describing the pain of NOT using the product.
4. **Features as Benefits**: 3 features, but write them entirely as benefits (What's in it for the user?).
5. **Final CTA Section**: A compelling closing argument and a final CTA button text.
```

## Tips for Success

- **Be specific in the context**: The more detailed your inputs are in the brackets `[]`, the better the output will be.
- **Iterate**: Don't accept the first output. Reply with, "Make the hero headline punchier and more urgent," or "Focus more on the time-saving aspect."
