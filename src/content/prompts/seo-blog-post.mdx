# The Ultimate SEO Blog Post Prompt

Google is getting better at detecting generic AI content. To rank in 2024, your AI-generated content needs structure, entities, and a distinct voice.

## The Prompt

```text
Act as a Senior SEO Content Writer. I need you to write a 1,500-word blog post about "[Your Topic]".

Before writing, please understand the following guidelines:
1. **Target Keyword**: [Primary Keyword]
2. **Secondary Keywords**: [List 3-5 keywords]
3. **Tone**: Conversational, authoritative, but not overly formal. Use short paragraphs.
4. **Structure**: 
   - An engaging introduction using the PAS (Problem-Agitation-Solution) framework.
   - A clear H2 for the main definition.
   - At least 3 actionable subsections (H3s) with step-by-step instructions.
   - A conclusion with a clear call-to-action (CTA).

**Crucial Instruction**: Do NOT use generic AI transition phrases like "In conclusion," "Moreover," "It's important to note that," or "In today's fast-paced digital world." Write like a human expert.

Please output the blog post in Markdown format.
```
